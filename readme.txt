# Filtro ricerca categorie WordPress

**Versione:** 1.1  
**Autore:** Silvio Ricci

## Descrizione

Questo plugin aggiunge un campo di ricerca per filtrare le categorie nell’editor classico e nella pagina di gestione delle categorie di WordPress. Facilita la selezione e la gestione delle categorie quando sono presenti molte opzioni, migliorando l’usabilità del pannello di amministrazione.

## Funzionalità

- **Ricerca nelle categorie (Editor Classico):**  
  Aggiunge un campo di ricerca sopra la lista delle categorie nell’editor classico degli articoli. Filtra le categorie in tempo reale mentre digiti.

- **Ricerca nel menu a tendina della categoria genitore:**  
  Nelle pagine di gestione delle categorie, aggiunge un campo di ricerca sopra il menu a tendina per selezionare la categoria genitore, permettendo di filtrare rapidamente l’elenco.

- **Pulsante cancella ("×"):**  
  Ogni campo di ricerca include un pulsante per cancellare facilmente il testo inserito e visualizzare tutte le opzioni disponibili.

- **Interfaccia intuitiva:**  
  Tutto è integrato nell’interfaccia di amministrazione di WordPress, senza influire sul sito pubblico.

## Installazione

1. Scarica il file `categoria-search-filter.php`.
2. Carica il file nella cartella `/wp-content/plugins/` del tuo sito WordPress.
3. Attiva il plugin dalla sezione “Plugin” della Bacheca di WordPress.

## Utilizzo

- **Editor Articoli:**  
  Troverai il campo di ricerca sopra la lista delle categorie nella sidebar dell’editor classico.

- **Gestione Categorie:**  
  Quando aggiungi o modifichi una categoria, troverai il campo di ricerca sopra il menu a tendina “Categoria genitore”.

## Note tecniche

- Il plugin funziona solo nell’area amministrativa di WordPress.
- Utilizza jQuery per il filtraggio in tempo reale.
- Tutti i testi e le istruzioni sono in italiano.
